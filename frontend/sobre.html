Sobre o Projeto

Este projeto começou como um trabalho para a disciplina de Introdução à Computação, focado em HTML e CSS. Após a conclusão do curso, decidi expandir a ideia de forma independente, transformando-o em um projeto pessoal para aprofundar meus conhecimentos em desenvolvimento web.

GerIA é um gerador de imagens que utiliza inteligência artificial. Os usuários podem descrever a imagem que desejam, e a aplicação a cria em tempo real. Além disso, as imagens geradas são salvas em uma galeria persistente, onde podem ser visualizadas e baixadas.

por:

  • Bárbara Cabral

Tecnologias utilizadas:

  • HTML
  • CSS
  • JavaScript
  • Python
  • Flask
  • API (OpenAI)
  • SQLite

Nota: A funcionalidade de geração de imagens e a galeria não operam no GitHub Pages, rodando apenas localmente, porque o backend, que protege a chave de API e salva as imagens, não está em um servidor público. Para que o projeto funcione online, o backend precisa ser migrado para uma plataforma de hospedagem. A implantação dessa migração é o próximo passo planejado para o projeto.
